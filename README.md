# 🏪 Vyapar Sahayak — AI Business Assistant

An AI-powered business management assistant for Indian small businesses, built with Python, Flask, and Groq AI.

## 🚀 Live Demo
👉 [vyaparsahayak.onrender.com](https://vyaparsahayak.onrender.com)

## ✨ Features
- 💬 **AI Chat** — Hindi/Hinglish mein baat karo
- 📦 **Stock Tracking** — Automatic stock management
- 🧾 **Invoice Generation** — GST-ready invoices
- 📊 **Sales Dashboard** — Real-time analytics
- 💾 **Database** — Supabase powered storage
- 📱 **WhatsApp Integration** — Wati.io connected

## 🛠️ Tech Stack
| Technology | Purpose |
|------------|---------|
| Python + Flask | Backend API |
| Groq AI (LLaMA 3.3) | AI Engine |
| Supabase | Database |
| Render | Cloud Deployment |
| Wati.io | WhatsApp API |
| HTML/CSS/JS | Frontend |

## 🚀 Setup
```bash
git clone https://github.com/omkanojiya460-oss/VyaparSahayak
cd VyaparSahayak
pip install -r requirements.txt
python app.py
```
